Project Overview: Moallamyar (Teacher-Assistant) is a specialized AI agent developed to empower educators by providing real-time pedagogical support. It is specifically tuned to understand the unique educational landscape and cultural context of Iranian schools.

Technical Implementation & Features:

Context-Aware Dialogues: Implements a state-managed ConversationHandler to collect critical context such as grade level (Elementary to University), subject matter, and specific classroom challenges before providing advice.

Customized Prompt Engineering: Features a sophisticated prompt system that instructs the AI to act as a compassionate and professional consultant, considering the local educational requirements and socio-economic factors of Iran.

Dynamic Session Management: Built with an asynchronous architecture using python-telegram-bot, allowing teachers to easily restart sessions or update their profiles via the /restart command.

Gemini Pro Integration: Utilizes advanced LLM capabilities to generate diverse content, from lesson plans and creative teaching methods to evaluation strategies and digital content production ideas.

Asynchronous Processing: Ensures high responsiveness and the ability to handle multiple teacher requests simultaneously without latency.

Technical Stack:

AI Engine: Google Generative AI (Gemini Pro).

Framework: Python Telegram Bot (Async API).

Language: Python 3.x.

Logic: State-Machine based conversation flows.
